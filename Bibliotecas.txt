
Faker

   A biblioteca `Faker` é uma ferramenta popular para gerar dados fictícios de maneira simples e flexível.
Ela é útil para criar dados de teste que imitam informações do mundo real, como nomes, endereços, e-mails, números de telefone, 
URLs e muito mais. Isso é especialmente valioso em situações onde você precisa testar seu código com dados variados, 
mas não quer usar informações reais por questões de privacidade ou segurança.
   Com `Faker`, você pode criar dados de teste de maneira rápida e eficiente.
Por exemplo, você pode gerar um nome aleatório usando `faker.name()`, um endereço com `faker.address()`, 
ou uma URL com `faker.url()`. A biblioteca suporta múltiplos idiomas e formatos, 
tornando-a uma escolha versátil para desenvolvedores que precisam de dados de teste variados.


Unittest

  `unittest` é a biblioteca de testes padrão do Python, fornecendo um framework para escrever e executar testes automatizados. 
Ela permite que os desenvolvedores criem testes unitários, que são testes que verificam pequenas partes de um programa 
(geralmente funções ou métodos) de forma isolada.
   Com `unittest`, você pode organizar seus testes em classes, definir métodos de teste que começam com `test_`, 
e usar uma variedade de asserções (como `assertEqual`, `assertIsNone`, etc.) para verificar se o comportamento do seu código está 
correto. A biblioteca também suporta a execução de testes de forma automática e fornece relatórios detalhados sobre a execução dos 
testes, incluindo informações sobre falhas e erros. Isso ajuda os desenvolvedores a garantir que seu código funcione conforme o 
esperado à medida que novas funcionalidades são adicionadas ou alterações são feitas.
